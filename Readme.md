# Milves


# Windows

### 配置虚拟环境
```bash
> cmd
> cd D:\devtools\PythonVenv
> python3 -m venv milves
> D:\devtools\PythonVenv\milves\Scripts\activate.bat
```


## 安装依赖

```bash
> pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```




## 服务启动

```bash
> sudo docker compose up -d
```
